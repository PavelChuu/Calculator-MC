# Calculator-MC
Обычный калькулятор расходов и калорий.
 
<b/>Что они умеют:</b><br/>
*  Калькулятор расходов:<br/>
    -  Сохранять новую запись о расходах<br/>
    -  Считать, сколько денег было потрачено сегодня<br/>
    -  Определять, сколько ещё денег можно потратить сегодня<br/>
    -  Считать, сколько денег было потрачено за 7 дней<br/>
*  Калькулятор калорий:<br/>
    -  Сохранять запись о приёме пищи<br/>
    -  Считать, сколько калорий было съедено сегодня<br/>
    -  Определить, сколько калорий ещё можно/нужно съесть сегодня<br/>
    -  Считать, сколько калорий было съедено за 7 дней<br/>
